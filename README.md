# Curriculum vitae website in React

Dit project is een interactieve CV-website, gebouwd met React.
Het doel is om een overzichtelijke en gebruiksvriendelijke online CV te presenteren.
Voor deployment gebruik ik Vercel. U vind de live versie hier;
https://cv-kara-van-de-velde.vercel.app/

## Dependencies
-React 19.0.0
-react-dom 19.0.0
-React Router 7.2.0 (voor navigatie)
-fontawesome-free 6.7.2 (voor iconen)

## Development tool
-Vite 6.1.0 (voor snelle build)

## Features
-Mobile first design
-Navigatie via React Router
-Dark mode toggle
